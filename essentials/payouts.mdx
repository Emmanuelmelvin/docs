# Payouts

Payouts allow you to withdraw funds from your Orafi balance to external accounts.

## Payout Methods

- **CRYPTO**: Withdraw to cryptocurrency wallets
- **FIAT**: Withdraw to bank accounts

## Checking Balances

Get available balances:

```javascript
const response = await fetch('/transactions/payout/balances', {
  headers: {
    'x-api-key': process.env.ORAFI_API_KEY
  }
});

const balances = await response.json();
console.log('Crypto balance:', balances.data.crypto);
console.log('Fiat balance:', balances.data.fiat);
```

## Creating Crypto Payouts

```javascript
const payout = await fetch('/transactions/payout/crypto', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': process.env.ORAFI_API_KEY
  },
  body: JSON.stringify({
    amount: 5000, // Amount in USDC
    walletAddress: '0x742d35Cc6634C0532925a3b844Bc454e4438f44e'
  })
});
```

## Retrying Failed Payouts

```javascript
await fetch('/transactions/payout/crypto/retry', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': process.env.ORAFI_API_KEY
  },
  body: JSON.stringify({
    payoutId: 'payout_abc123'
  })
});
```

## Best Practices

1. **Settlement accounts**: Set up accounts before payouts
2. **Balance checks**: Verify sufficient balance before payout
3. **Address verification**: Double-check wallet addresses
4. **Fee awareness**: Account for network fees
5. **Confirmation waiting**: Allow time for blockchain confirmation
