# Authentication

Orafi uses API keys for authenticating requests to the public API. For dashboard operations, JWT tokens are used.

## API Key Authentication

Include your API key in the `x-api-key` header for all API requests.

### Example

```bash
curl -X GET https://api.orafi.com/transactions/payment/retrieve?payment_id=pay_123 \
  -H "x-api-key: ora_live_1234567890abcdef"
```

### API Key Format

- **Live keys**: `ora_live_` followed by 32 characters
- **Test keys**: `ora_test_` followed by 32 characters

Keep your API keys secure. Never expose them in client-side code or public repositories.

## JWT Authentication

For dashboard operations (user management, profile settings), use JWT tokens obtained from login.

Include the token in the `Authorization` header with `Bearer` prefix.

### Example

```bash
curl -X GET https://api.orafi.com/profile \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

## Security Best Practices

1. **Use HTTPS**: Always make requests over HTTPS
2. **Secure storage**: Store API keys securely, never in version control
3. **Key rotation**: Rotate keys regularly
4. **Environment separation**: Use test keys for development
5. **Webhook verification**: Always verify webhook signatures

## Example Requests

### Creating a payment with API key

```javascript
const response = await fetch('https://api.orafi.com/transactions/payment/hosted/create', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'ora_live_1234567890abcdef'
  },
  body: JSON.stringify({
    amount: 5000,
    customer: {
      fullname: 'Jane Smith',
      email: 'jane@example.com'
    },
    txRef: 'order_987654',
    type: 'HOSTED_CHECKOUT'
  })
});
```

### Node.js with API key

```javascript
const axios = require('axios');

const client = axios.create({
  baseURL: 'https://api.orafi.com',
  headers: {
    'x-api-key': 'ora_live_1234567890abcdef',
    'Content-Type': 'application/json'
  }
});

const payment = await client.post('/transactions/payment/hosted/create', {
  amount: 2500,
  customer: {
    fullname: 'Alice Johnson',
    email: 'alice@example.com'
  },
  txRef: 'inv_456789',
  type: 'HOSTED_CHECKOUT'
});
```
