- GET /settlement/fiat/bank-list - Getting Bank List
- POST /settlement/fiat/add - Adding Fiat Accounts
- GET /settlement/fiat - Get Fiat Accounts
- PATCH /settlement/fiat/{id} - Update Fiat Account
- DELETE /settlement/fiat/{id} - Delete Fiat Account


## Fiat Settlements

### Getting Bank List

```javascript
const response = await fetch('/settlement/fiat/bank-list', {
  headers: {
    'x-api-key': process.env.ORAFI_API_KEY
  }
});
```

### Adding Accounts

```javascript
await fetch('/settlement/fiat/add', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': process.env.ORAFI_API_KEY
  },
  body: JSON.stringify({
    bankName: 'Bank of America',
    bankCode: 'BOA',
    accountNumber: '1234567890'
  })
});
```

### Managing Accounts

```javascript
// Get accounts
const response = await fetch('/settlement/fiat', {
  headers: {
    'x-api-key': process.env.ORAFI_API_KEY
  }
});

// Update account
await fetch(`/settlement/fiat/${accountId}`, {
  method: 'PATCH',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': process.env.ORAFI_API_KEY
  },
  body: JSON.stringify({
    bankName: 'Updated Bank Name'
  })
});

// Delete account
await fetch(`/settlement/fiat/${accountId}`, {
  method: 'DELETE',
  headers: {
    'x-api-key': process.env.ORAFI_API_KEY
  }
});
```

## Best Practices

1. **Verification**: Verify account ownership
2. **Security**: Use secure account details
3. **Multiple accounts**: Set up backup settlement accounts
4. **Currency matching**: Use appropriate accounts for currencies
5. **Regular updates**: Keep account information current
